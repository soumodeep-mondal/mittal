Hologram

Hologram is a 3D image security concept produced by an interference pattern of laser light. Through a cutting-edge technology called ‘Laser optics’, a Hologram is created - it’s a three-dimensional, multicolored laser image, which can not be transferred from one surface to another. Further, the addition of covert security devices, sequential numbering, Hidden Text, Micro Text, tamper evident adhesives and other security features make a well-produced security Hologram, which can not be duplicate by anyone.
